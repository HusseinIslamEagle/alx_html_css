## üìñ About The Project

**Headphones** is a responsive landing page project developed as part of the ALX HTML & CSS curriculum. The main objective was to build a visually engaging webpage using advanced CSS techniques and Vanilla JavaScript, ensuring it is responsive across all devices without relying on external frameworks like Bootstrap.

The project highlights the use of **CSS Shapes** (Pentagons), **Custom Animations**, and a **JavaScript-powered Hamburger Menu**.

---

## ‚ú® Key Features

* üì± **Fully Responsive:** Fluid design that adapts seamlessly from desktop to mobile screens.
* üçî **Hamburger Menu:** A custom sidebar navigation for mobile devices, built with Vanilla JS.
* üé® **CSS Art:** The "Our Results" section features pentagons drawn entirely using CSS `clip-path` (no images used).
* üé≠ **Interactive Animations:** * Smooth hover effects on cards and buttons.
    * Bouncing and rotating animations for the pentagons.
    * Icon scaling effects in the "What we do" section.

## üõ† Technologies Used

* **HTML5**: Semantic structure of the webpage.
* **CSS3**: Flexbox, Media Queries, CSS Variables, Transitions, Transforms, and Clip-path.
