## üìñ About The Project

**Smile School** is a web development project that focuses on implementing a design from a Figma file into a fully functional, responsive webpage. The goal was to create a pixel-perfect replica using only **HTML5** and **CSS3**, without relying on frameworks like Bootstrap.

The project features a modern layout using **Flexbox**, custom fonts (Google Fonts), and interactive hover effects.

### Key Features
* ‚ú® **Responsive Design:** Looks great on desktops, tablets, and mobile devices.
* üé® **Modern Layout:** Built entirely using CSS Flexbox for structure.
* üñåÔ∏è **Pixel Perfect:** Meticulously styled to match the provided Figma design.
* üé≠ **Interactive Elements:** Smooth hover effects on buttons, cards, and navigation links.
* ‚úíÔ∏è **Custom Typography:** Integrated Google Fonts (*Coiny* & *Source Sans Pro*).

## üöÄ How to Run

You can view this project locally on your machine.

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/HusseinIslamEagle/alx_html_css/tree/main/css_advanced](https://github.com/HusseinIslamEagle/alx_html_css/tree/main/css_advanced)
    ```

2.  **Navigate to the project directory**
    ```bash
    cd smile-school
    ```

3.  **Open `index.html`**
    * Simply double-click the `index.html` file to open it in your default browser.
    * OR use a Live Server extension in VS Code.
